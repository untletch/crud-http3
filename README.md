# Golang http3 client server

## About The Project
A simple golang http3 client server

## Clone the Repo

```sh
git clone https://github.com/untletch/crud-http3.git
```

## Run the app locally

```sh
cd crud-http3
# for http3 server
make server
./bin/server
# for http3 client
make client
./bin/client -help
./bin/client -url https://codepen.io
```
